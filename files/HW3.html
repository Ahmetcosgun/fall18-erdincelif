Using ECG spectra of 100 train and 100 test data, sick and healthy people are classified as -1 and 1, respectively. First, these classes are converted to binary classes 0 and 1.

In fused Lasso calculations, lambda 2 is set to 0 and best value for lambda 1 is calculated using optL1 function. Here is the optimization results for lambda 2:

Using penalized function, predictions on test data is performed. Then, test predictions and test results are compared using confusion matrix. In Figure 12 confusion matrix and its statistics are given. Accuracy on test data is found as 0.83.

There are 97 regression coefficients in this task. Only 14 of those are found non-zero. Zero coefficients means that those variable has no effect on the prediction model. The main reason may be that zero coefficient parameters has no variation. Regarding the ECG spectra, for some time values spectra may be on its baseline and may not be showing any variation.

2.2. Task 2.c & Task 2.d

In this part, delta spectra is calculated for delta time. Similarly, logistic regression model is created on this data. Same methodology is followed, first optimal lambda 2 values is calculated as in Figure 13.

Again, using the optimal lambda 2 value, logistic regression model with fused Lasso is applied on the transformed data. **Figure 14 summarizes the confusion matrix and its statistics. Accuracy of the model on the test data is found as 0.78. Among 96 regression parameters, only 19 of them are found as non-zero.

When the penalized regression on raw and transformed data are compared, it is seen that transforming the data by taking time variant differences reduced the accuracy of the model. Besides, in the second model, the number of zero coefficient parameters are increased. One of the reasons behind this may be the reduction in the variables from 97 to 96. However, I do not think its effect is major. The main reason for the reduced accuracy may come from the decrease in the variance of the spectra data. By taking stepwise differences in the spectra, variations in time is diminished.
